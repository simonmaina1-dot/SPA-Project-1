# Route Map

- `/` -> Home (hero + stats + featured + search/filter/sort + projects grid + about)
- `/projects` -> Home alias (same content, keeps SPA navigation consistent)
- `/projects/:id` -> Project details page (fallback for deep-links/bookmarks)
- `/donate/:id` -> Donation flow (mock checkout)
- `/dashboard` -> Admin dashboard (create/update projects, status toggle, image preview)
